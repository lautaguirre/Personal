MI TRABAJO
Villa Martina Consultorios

Este proyecto fue creado enteramente por mi, se trata de una web con un diseño responsivo que se encarga de gestionar reservas de consultorios por hora para que ciertos profesionales puedan desarrollar sus actividades. Los clientes, luego de loguearse con su cuenta, pueden ver mediante un calendario la disponibilidad horaria de las oficinas y realizar la reserva correspondiente a sus intereses mediante una interfaz muy intuitiva. Tambien cuenta con un panel de admin donde los propietarios pueden crear, actualizar, inhabilitar y consultar los datos de los usuarios como asi tambien las reservas realizadas en cada mes.

Para desarrollar este sistema web utilice las siguientes herramientas:

  • Front-end: HTML5 y CSS3 con Bootstrap 3 junto a Javascript con jQuery
  • Intercambio de datos: Ajax con JSON
  • Back-end: PHP
  • Base de datos: MySQL
  • Librerias: Fullcalendar para la interfaz del calendario y Moment.js para el manejo de fechas.
